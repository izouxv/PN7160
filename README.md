# PN7160 ESP Projects

This repository contains ESP-based projects for the **NXP PN7160 NFC controller**, mainly used for feature verification and development reference.

Included projects:

- **ESP-PN7160-READ-CARD**  
  Demo project for testing and verifying NFC card reading using PN7160.

- **ESP-PN7160-UPDATE-FIRMWARE**  
  Project for updating PN7160 firmware via an ESP host controller.

This repository is intended for PN7160 driver development, functional testing, and project reference.
